
## To Run Locally

Clone the project

```bash
  git clone https://github.com/piyush-agrawal6/Hospital-Management-System.git
```

Go to the project directory

```bash
  cd Hospital-Management-System
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
